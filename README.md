# Calculadora de Preço de Impressão 3D

Pronta para publicar no **GitHub Pages**.

## Como publicar no GitHub

1. Entre em [github.com](https://github.com) e crie um repositório novo, por exemplo `calculadora-3d`.
2. Faça upload dos ficheiros desta pasta para a raiz do repositório.
3. No GitHub, vá a **Settings > Pages**.
4. Em **Build and deployment**, escolha:
   - **Source:** Deploy from a branch
   - **Branch:** `main`
   - **Folder:** `/ (root)`
5. Guarde.
6. Aguarde 1 a 3 minutos. O GitHub vai mostrar o link público, algo como:
   `https://SEU-USUARIO.github.io/calculadora-3d/`

## Como instalar no iPhone

1. Abra esse link no **Safari**.
2. Toque em **Partilhar**.
3. Toque em **Adicionar ao Ecrã principal**.
4. Confirme o nome e toque em **Adicionar**.
